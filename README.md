## 🧠 What Is This?

This bot integrates the [Stockfish](https://stockfishchess.org/) chess engine with Chess.com to provide **instant best-move suggestions** as you play. It reads the current board state and tells you exactly where to move — no more guessing, no more blunders.

Built in Python. Simple to set up. Devastatingly effective.

---

## ✨ Features

- ⚡ **Real-time analysis** — evaluates positions as the game unfolds
- 🏆 **Stockfish-powered** — one of the strongest chess engines in the world
- 🐍 **Pure Python** — easy to read, extend, and hack on
- 🔄 **Actively developed** — new improvements ship with every release (see [Changelog](#-changelog))

- Python 3.8+
- [Stockfish](https://stockfishchess.org/download/) installed on your system
